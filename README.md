## Installation

### Docker Image:

[crazy0x/vue-nga:1.4](https://hub.docker.com/r/crazy0x/vue-nga/tags/)

***

### Debug

runtime environment

- Node.js 4+
- npm 3+
- Python 2.7.x

安装 cooking
```

  npm i cooking-cli -g

```

- 部署server端
- 修改./vuex/store.js接口地址，指向后端部署的URL
- 进入项目目录

```

  npm install
  cooking watch

```
- 浏览器（调试模式中选择移动设备），打开主页 127.0.0.1:8808/home

***

## Building

```
  cooking build
```
